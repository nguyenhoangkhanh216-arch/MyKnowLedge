## Summary

Nguyễn Hải Nam (Engineering Manager, PTSC M&C) issued EDMS workflow guidance on 26-Jun-2026 for TBE Category 2 documents. Workflow WF03B is to be used; TBE CAT2 documents are issued as AFP (Approved For Procurement) with revision numbers (e.g., Rev0). Step-by-step workflow ownership assignments were clarified.

## Key Points

- **EDMS system:** Internal Electronic Document Management System for KMDD project
- **Workflow for TBE CAT2:** Use **WF03B** (not WF03)
- **Document status:** AFP (Approved For Procurement)
- **Revision format:** Use numeric revision numbers (e.g., Rev0)

### Workflow step ownership:
- **Steps 3, 4/8 and Reject step:** Package Engineer (PIC) — e.g., `namnh3` (package engineer for the relevant package)
- **Steps 1, 5/8:** Discipline Lead (approval at discipline level) — e.g., `namnhB` as lead
- **EM approval and DCC:** Already set as default — no action needed

### Workflow tip (Step 4/8 re-issue):
- When re-issuing TBE Rev AFP, use the attachment function specifically for this step

### Historical context:
- WF03: used for IFR/IFA/IFC document issuance to client; only 1 PIC as Lead to Check (assigning 2 PICs breaks the workflow)
- WF02 → WF01A: for IDC documents; use WF01A if additional reviewer needed after WF02 started
