## Summary

Master Equipment Lists (MEL) for both KM-WHP and DD-WHP were issued on 15-Jun-2026 for Interdiscipline Check (IDC), assigned document code Code 2. Issued by Nguyễn Sử Trần Hoàng (PTSC M&C). Equipment tags follow KDP-OOO-00-PS-G-0003 Rev B2 numbering convention (confirmed same day by PVEP-KM).

## Key Points

- **Document issued:** KDP-KMA-03-LI-M-0001-D0 (KM-WHP MEL) — Code 2 for IDC
- **Document issued:** KDP-DDA-03-LI-M-0001-D0 (DD-WHP MEL) — Code 2 for IDC
- **Review type:** IDC (Interdiscipline Check) — internal multi-discipline review milestone; all engineering disciplines must review and comment
- **Issued by:** Nguyễn Sử Trần Hoàng, PTSC M&C
- **Tag numbering:** Per KDP-OOO-00-PS-G-0003 Rev B2 (Equipment Tagging and Numbering Procedure)
  - "01" in tag = KM-WHP equipment (e.g., RLR-81-**01**01)
  - "02" in tag = DD-WHP equipment (e.g., RLR-81-**02**01)
- **Code 2 meaning:** Approved with comments — multi-discipline review required before IFA/IFR issue

### IDC Process
- IDC is an internal engineering gate: all disciplines (Process, Piping, Mechanical, Instrument, Electrical, Structural, Civil) review documents at the same time
- Comments consolidated by document owner
- After IDC close-out, document progresses to IFR (Issued for Review by CLIENT) or IFA (Issued for Approval)
