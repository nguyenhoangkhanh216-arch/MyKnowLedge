## Summary
A significant scope change was issued for INS-002 Pressure Relief Valves on Jun 3: HIPPS valve leakage PSVs are now reclassified as Base Scope (previously Optional Scope). Four (4) HIPPS PSVs added to Tag-list 01. All bidders must re-size and resubmit datasheets and GA drawings. Response deadline 15:45 Jun 4 — making this the most urgent action item for today.

## Key Points
- **Scope change**: HIPPS PSV leakage moved from Optional to **Base Scope**.
- **Quantity**: 4 nos. HIPPS PSVs added to Tag-list 01.
- **Action for bidders**: Receive latest PR data and process datasheets → re-size PSVs → submit new DS and GAD.
- **Deadline: 15:45 Jun 4** (today) — issued by Nguyễn Anh Khoa (Instrument Engineer).
- **Background on CA change**: Earlier (20-May), CA requirement was upgraded to CAT C and made Base Scope — requires ABS, LR, DNV, or BV for overseas manufacture; Vietnam Register for local.
- **Bidders affected**: TTE, An Phu Tech, Thanh An, ATA, KLE — all received TBC#2 Rev.B on Jun 3.
- **TBC revisions chain**: Started TBC#1 RevA (Apr 29) → multiple revisions → TBC#2 Rev.B now includes HIPPS PSV as base scope.
