## Summary

Appendix H2 Rev A (11 pages) defines the material and equipment certification requirements for the NLQ EPCIC 10 project. It establishes 5 certification types (A through E) based on EN 10204 and specifies which type applies to each material or equipment category via the Certification Matrix (Table 6-1).

## Key Points

### Certification Types — EN 10204 Based
- Type A (EN 10204 type 3.2): CA-endorsed compliance certificate — highest level; applies to CRA materials, sour service, lethal service, Special/First Category steel, riser piping, pipeline components — Exhibit H App H2, §4
- Type B (EN 10204 type 3.1): Vendor-issued compliance certificate with test results defined by product spec; Second Category structural steel, instruments, welding consumables — Exhibit H App H2, §4
- Type C (EN 10204 type 2.2): Vendor-issued certificate based on routine production tests — Exhibit H App H2, §4
- Type D (EN 10204 type 2.1): Vendor compliance certificate without test results — Exhibit H App H2, §4
- Type E: Vendor-issued compliance certificate — used for mechanical/electrical equipment alongside C and D — Exhibit H App H2, §4
- Stockist certificates and "typical" certificates are not acceptable — Exhibit H App H2, §4.1

### Documentation Quality Rules
- All certificates must be typed/printed — handwritten unacceptable — Exhibit H App H2, §4.2
- All certificates must be in English; notarised translation if not possible — Exhibit H App H2, §4.2
- No additions, deletions, or corrections; correction fluid unacceptable — Exhibit H App H2, §4.2

### Key Specific Certification Requirements
- CRA/Duplex/Super-Duplex/Sour service/Lethal service materials: Type A — Exhibit H App H2, §6 Table 6-1
- First and Special Category structural steel: Type A — Exhibit H App H2, §5.2.1
- Second Category structural steel: Type B — Exhibit H App H2, §5.2.2
- Passive fire protection and fire-rated architectural items: fire test certificate + fire test report required (from nationally accredited lab or CA-witnessed prototype test) — Exhibit H App H2, §5.2.4
- Fire-safe valves: fire test certification per API 6FA, API 607, or BS 6755 Part II — Exhibit H App H2, §5.2.13
- ESD valves on risers and SSIVs: Type A for body/bonnet + CA type test/approval certificate — Exhibit H App H2, §6 Table 6-1
- PSVs and relief valves: type test per API 520; Type A or B for body/bonnet + calibration certificate — Exhibit H App H2, §5.2.14 and Table 6-1
- Pressure vessels: ASME VIII certificate of code compliance (Form X or Form U-1A/A-1) + CA suitability certificate — Exhibit H App H2, §5.2.15
- Lifting appliances ≥10t SWL: Type A/B certification + proof load test certificate + SWL tag — Exhibit H App H2, §5.2.16
- Cranes: critical welds re-NDT after Proof Load Test by TPI; NDT procedure APPROVED by COMPANY; before Sail Away — Exhibit H App H2, §5.2.19
- Living quarters lift/elevator: ISO 8383 compliance certified by CA — Exhibit H App H2, §5.2.18
- HVAC system components: all Criticality Rating 2 per App H4

### Release Certificate — Mandatory for All Equipment
- All equipment/material released from vendor must be accompanied by: CONTRACTOR's Inspection Release Certificate, CA Inspection Release Note (where applicable), Type D letter of conformity, material test certificates for weld items, Outstanding Work List — Exhibit H App H2, §5.1
