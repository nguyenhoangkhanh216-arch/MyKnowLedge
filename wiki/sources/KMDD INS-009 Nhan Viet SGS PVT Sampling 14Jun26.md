## Summary

Kate Nguyen (Nhan Viet ICS JSC) forwarded SGS's recommendation on 14-Jun-2026 23:01 for the PVT sampling & injection configuration on KMDD. PTSC Instrument Engineer Nguyễn Thanh Sơn had asked on 12-Jun whether the existing configuration is compatible with PVT connection; SGS (PVT Service) confirmed the tracer injection DBB valve is acceptable but the sampling point design must be revised (no quill, 1" NPT fullbore, top-mounted). SGS also recommended PVT data be collected for both allocation and well test flowmeters at first-round commissioning.

## Key Points

- **DBB valve for tracer injection:** Acceptable configuration for tracer injection chemical injection point
- **DBBV sampling point — no sampling quill:** SGS advises retractable probe instead; requires 1" NPT DBBV with fullbore minimum 14mm bore
- **Sampling point orientation:** Preferred top (12 o'clock) of horizontal pipe; bottom (6 o'clock) is acceptable only if well has no sand production (eliminates need for retractable probe)
- **PVT data at first round:** Both well test flowmeter AND allocation flowmeter need PVT (wellstream composition) data before handover to end-user production team
- **Subsequent calibration frequency:** Allocation flowmeter — more frequent PVT checks to maintain accuracy; Well test flowmeter — every 1–2 years (or longer) per end-user preference and reservoir performance
- **CEESI wet gas facility:** Nhan Viet (McCrometer V-Cone) could not get formal letter from CEESI for 8" loop without detailed test plan; McCrometer factory calibration by water with wet gas correction offered as alternative (confirmed sufficient for wet gas by McCrometer)
- **Context:** This resolves the open SGS configuration check; cost impact confirmation and PVT sampling configuration are the remaining open items on INS-009 before TBC close
