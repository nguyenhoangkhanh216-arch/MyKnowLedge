## Summary

Bao Nghi (via Kim Ngân) responded to two PTSC queries about DBBV installation for the INS-009 Allocation Flowmeter package: (1) vertical 3-o'clock orientation is acceptable for tracer injection; (2) the current 10mm full-bore DBBV with 1/2" NPT connection is NOT compatible with SGS's retractable sampling probe — SGS recommends 6-o'clock installation instead, but notes clogging risk if well produces sand.

## Key Points

- **Package:** KMDD-INS-009 — Allocation and Well Test Flowmeter
- **From:** Đặng Kim Ngân (Bao Nghi) — checked with manufacturer and SGS
- **Date:** 26-Jun-2026, 13:55

### Point 1 — Tracer Injection at 3-o'clock DBBV:
- PTSC asked if a DBBV at vertical 3-o'clock is suitable for tracer injection
- **Answer:** No issue. Must ensure accessibility (personnel can reach injection point) and sufficient space for gas and liquid injection skids nearby

### Point 2 — DBBV Sampling Point for SGS retractable probe:
- PTSC asked: if no sampling quill, how is pressure isolated during retractable probe installation?
- **Answer:** Isolation via the 2 handles (2 barriers) of the DBB valve itself — not from the sampling quill
- PTSC asked: current design has 10mm full-bore DBBV with 1/2" NPT(F) connection — can SGS use this?
- **Answer:** **No** — 10mm DBBV with 1/2" NPT(F) is NOT compatible with SGS's retractable sampling probe
- **SGS recommendation:** Install DBBV at 6-o'clock (bottom) of horizontal pipe
- **Risk note:** 6-o'clock installation risks clogging if the well produces sand over time
