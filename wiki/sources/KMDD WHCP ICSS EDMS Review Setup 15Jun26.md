## Summary

On 15-Jun-2026, Nam (PTSC M&C) forwarded the WHCP/ICSS package assignment matrix to set up EDMS review for vendor documents already provided by VSP (Vietsovpetro/XNCĐ). VSP had previously submitted P&ID, Hydraulic Calculations, and GA/3D Naviswork drawings for both KM-WHP and DD-WHP WHCP packages. - **Action:** Nam (PTSC M&C) forwarded package assignment matrix → EDMS review setup in progress
- **VSP deliverables previously received (both platforms):**
  - P&ID (WHCP — Wellhead Control Panel)
