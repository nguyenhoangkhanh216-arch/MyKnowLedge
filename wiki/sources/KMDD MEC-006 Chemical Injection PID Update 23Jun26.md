## Summary

Bùi Lê Lương (Mechanical Engineer, PTSC) raised an internal alert on 23-Jun-2026: P&ID Rev.E0 for the Chemical Injection Package contains two major undistributed updates that have not been shared with bidders. The changes affect all chemical injection system P&IDs (new PCV return loop) and the methanol injection system (all DBB valve sizing changed). This may require a formal TBC supplement or errata to be issued to all MEC-006 bidders.

## Key Points

- **Alert raised by:** Bùi Lê Lương (Mechanical), to Nguyễn Hồng Quân and Nguyễn Sử Trần Hoàng, 08:52 23-Jun
- **Document affected:** P&ID Rev.E0 — Chemical Injection Package
- **Change 1 — All chemical injection systems (PPD, SI, CI):**
  - A PCV (Pressure Control Valve) return system has been added to all chemical injection circuits
  - Many isolation valves and PCVs added as part of this return loop
  - This is a significant P&ID design change affecting all CI sub-systems
- **Change 2 — Methanol Injection system:**
  - Sizing of ALL DBB (Double Block and Bleed) valves has been changed
  - Affects the entire methanol injection skid valve specification
- **Status of distribution:** Not distributed to bidders — Bùi Lê Lương explicitly asked if team had ever distributed this information to bidders (answer implied: no)
- **Procurement impact:** If bidders have priced/designed to the previous P&ID revision, their technical proposals may be non-compliant with Rev.E0. A TBC amendment or formal addendum may be required before TBE can be completed
