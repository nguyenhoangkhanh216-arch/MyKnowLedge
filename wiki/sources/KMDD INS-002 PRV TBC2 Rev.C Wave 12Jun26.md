## Summary

On 12-Jun-2026 at 07:21-07:24, PTSC (Bảo) issued TBC#2 Rev.C (with updated Process DTS + supporting docs 11-Jun) simultaneously to all five INS-002 bidders: KLE (07:21), TTE (07:22), Thanh An/TAD (07:23), An Phu (07:24), ATA (07:24). Deadline for all: COB 15-Jun-26. Preceding this issuance, several bidders had responded to TBC#2 Rev.B, with An Phu flagging a metallurgical dispute about the ferrite content requirement on 11-Jun.

## Key Points

- **TBC#2 Rev.C issued to 5 bidders simultaneously** — 12-Jun 07:21-07:24; updated Process DTS + supporting docs 11-Jun enclosed; deadline COB **15-Jun-26** for all
- **Bidder pre-responses before Rev.C:**
  - KLE (minhnt@khanhlinhco.com.vn / Nguyen Thanh Minh) — responded TBC#2 Rev.B on 08-Jun with datasheets, drawings, cost impact, Form T5/T9
  - TTE (Nguyen Huu Tien) — responded TBC#2 + cost impact on 10-Jun
  - Thanh An + TAD Power — resubmitted Form T9 + TBC#2 Rev.B on 09-Jun; TBC#1 Rev.C gasket + lifting lug open (awaiting factory data)
  - An Phu (Lam Van Vu PM / Italian foundry Technical Srl) — responded TBC#2 Rev.B on 04-Jun; raised ferrite spec dispute on 11-Jun (see below)
  - ATA (Nguyen Quang Cuong) — acknowledged TBC#2 Rev.C receipt 12-Jun 07:29; committed to respond COB 15-Jun
- **Ferrite spec dispute (An Phu / Technical Srl, 11-Jun):** Item IN2-Rev B — ferrite requirement 45-55% flagged as: (1) narrower than standard 35-55%; (2) metallurgically contradicts high toughness requirements at low temperature (Italian foundry argues 40-42% ferrite is needed for better toughness). An Phu requested PTSC to forward to Customer (CPY) for clarification. PTSC's response was to issue TBC#2 Rev.C with updated Process DTS — unclear if updated DTS resolves the ferrite dispute.
- **Package scope background:** 5 bidders; CA Category B; Trillium Flow-France is ATA's sub-supplier; HIPPS leakage PSV tags added to base scope Jun 3
