## Summary

The KMDD ELE team (Nguyễn Thành Phúc) is gathering power consumption data for the HVAC Heating Control Panel (HCP) from bidders DYNAC and MOTEC for use in the platform electrical power load assessment. MOTEC's submission was more complete than DYNAC's. This is part of MEC-009 (HVAC Package) technical bid clarification.

## Key Points

- **Purpose:** HCP power consumption data is needed for the platform's electrical power load assessment (load list update)
- **Bidders:**
  - **MOTEC:** Provided more complete HCP power data (named as preferred data source by ELE team)
  - **DYNAC:** Data received but less complete
- **ELE team action:** Phúc sending comments on MOTEC data — verifying values and format before incorporation into load list
- **Standard request wording (for TBC):** "Bidder is requested to advise the power consumption of the HCP for Electrical Team review and consideration in the platform power load assessment"
- **Cross-discipline impact:** HVAC → ELE load list → affects UPS sizing, switchboard capacity for both KM-WHP and DD-WHP
